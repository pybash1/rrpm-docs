### `[root]`&nbsp; Section
- &nbsp;`dir`&nbsp;: String that determines the location of where to store projects.
- &nbsp;`ext_dir`&nbsp;: String that determines the location of where to look for extensions.
<br />
<br />
### `[cli]`&nbsp; Section
- &nbsp;`display_output`&nbsp;: Boolean that determines if the raw output of the commands ran or shown or not.
- &nbsp;`ignore_extension_load_error`&nbsp;: Boolean that determines whether to ignore extension loading errors or not.
<br />
<br />

### `[extensions]`&nbsp; Section
- &nbsp;`presets`&nbsp;: List of strings where each determines the file name of a preset extension.
- &nbsp;`hooks`&nbsp;: List of strings where each determines the file name of a hook extension.
<br />
<br />
