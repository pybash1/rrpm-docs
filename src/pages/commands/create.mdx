##### Usage
<Code code={`
Usage: python -m rrpm create [OPTIONS] NAME
  Generate a project from any of the presets and/or its variations
Arguments:
  NAME  [required]
Options:
  --src / --no-src  [default: no-src]
  --help            Show this message and exit.`} />

- &nbsp;`[NAME]`&nbsp;: Name of the project. Required for all presets.
- &nbsp;`src`&nbsp; / &nbsp;`--no-src`&nbsp;: Applicable for poetry projects only
