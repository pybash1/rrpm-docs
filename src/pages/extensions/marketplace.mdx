RRPM now has an official  marketplace for its extensions, right on this website. You can find it [here](/marketplace)!
You can copy the command to install extensions from here but cannot directly install them from here yet! To add your extension to the marketplace, add the "rrpm-extension" tag to your github repository, and the website should pick it up soon.
